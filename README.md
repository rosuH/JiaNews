## 简介

这是一个展示[嘉应学院官网](www.jyu.edu.cn)新闻动态的 Android App。

### 技术细节

- APP： Kotlin + RxJava + Retrofit
- 数据： Python 编写的爬虫获取
- 服务端： Flask + Gunicorn + Nginx
- [API](https://rosuh.me/jyu-api/)：OpenAPI3.0 + Restful
